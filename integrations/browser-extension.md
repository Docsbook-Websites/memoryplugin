# Browser extension

The fastest path for web AI tools. The extension works natively on ChatGPT, Claude, Gemini, Grok, DeepSeek, Perplexity, Mistral, Poe, LibreChat, and more, on Chrome and Safari, including mobile.

## How it works

Open any supported AI tool and click the MemoryPlugin button to inject the memories relevant to the conversation. Save new memories the same way - with a click or automatically.
