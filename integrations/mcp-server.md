# MCP server

For desktop and coding tools that speak the Model Context Protocol - Claude Desktop, Cursor, and Windsurf.

## How it works

Ask the AI to remember something and it flows into your shared memory; ask it to recall and the memory comes back. No separate button - it is part of the conversation.

## Who it is for

Coding workflows benefit most: save your stack and conventions once, and every MCP-connected tool starts with them. See [use cases](../use-cases.md).
