# Open API

For developers who want to build on the memory layer directly.

## What the API covers

- **Memory management** - create, update, delete, and bulk operations.
- **Chat history** - upload, search, recall, and list conversations.
- **Buckets** - create and manage the buckets that organize memories.

## Getting your token

Your authentication token is available in your dashboard after you create an account - see [quick setup](../getting-started.md).
