# Integrations

MemoryPlugin meets you where you already work. Choose one integration or combine several - the memory is shared across all of them.

## Browser extension

The fastest path for web AI tools. It works natively on ChatGPT, Claude, Gemini, Grok, DeepSeek, Perplexity, Mistral, Poe, LibreChat, and more, on Chrome and Safari, including mobile. Click the MemoryPlugin button to inject relevant memories into the conversation.

## MCP server

For desktop and coding tools that speak the Model Context Protocol - Claude Desktop, Cursor, and Windsurf. Ask the AI to remember something and it flows into your shared memory; ask it to recall and the memory comes back.

## Custom GPT

If your work lives inside ChatGPT, the Custom GPT integration brings your MemoryPlugin memory into that workflow directly.

## TypingMind plugin

Install the plugin and start immediately - no separate configuration step.

## Open API

For developers who want to build on top of the memory layer, the open API exposes memory management (create, update, delete, bulk operations), chat-history operations (upload, search, recall, list), and bucket management. Use it to wire MemoryPlugin into your own tools and agents.
