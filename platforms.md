# Supported platforms

MemoryPlugin works across more than 21 AI tools. How you connect depends on the tool.

## Web tools (browser extension)

ChatGPT, Claude, Gemini, Grok, DeepSeek, Perplexity, Mistral, Poe, LibreChat, and more - on Chrome and Safari, including mobile. See [browser extension](integrations/browser-extension.md).

## Desktop and coding tools (MCP server)

Claude Desktop, Cursor, and Windsurf over the Model Context Protocol. See [MCP server](integrations/mcp-server.md).

## Inside ChatGPT and TypingMind

Custom GPT and the TypingMind plugin. See [Custom GPT & TypingMind](integrations/custom-gpt-and-typingmind.md).
