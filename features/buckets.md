# Buckets

Buckets group memories by context - work, personal, a specific project or client.

## Why buckets

When you talk to your AI inside a given context, MemoryPlugin draws from the right bucket. A client's preferences do not leak into unrelated work, and each project keeps its own memory.

## Shared buckets

On Pro plans, buckets can be shared so a team draws from the same memory.
