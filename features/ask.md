# Ask

Ask is a single interface to query everything MemoryPlugin holds for you.

## What you can query

- **Memories** - the facts and preferences you have saved.
- **Chat history** - your imported past conversations.
- **Files** - PDFs and documents you have uploaded.

Answers come with source attribution, so you can see where each one came from.
